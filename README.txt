== Description ==

Publish your [Froont](http://froont.com) projects as posts or pages.

Use placeholders and replace them with your post or page data.

== Installation ==

1. Download the plugin and upload the contents of `froont.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
2. Activate the plugin
3. Under `Settings - Froont` set your Froont API KEY.
